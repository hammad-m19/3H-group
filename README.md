# 3H Group Builders & Developers Website

A modern, animated website for 3H Group Builders & Developers featuring smooth animations, responsive design, and an interactive user experience.

## Features

- **Animated Logo**: The iconic 3H logo with house-like "H" design animates on page load
- **Smooth Scroll Animations**: Elements fade in and animate as you scroll
- **Interactive Navigation**: Fixed navbar with smooth scrolling to sections
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI/UX**: Clean, professional design with smooth transitions
- **Performance Optimized**: Throttled scroll events and efficient animations

## Color Scheme

- **Red**: #E63946 (Primary accent color)
- **Blue**: #1E3A8A (Secondary accent color)
- **White**: #FFFFFF (Background)
- **Dark Gray**: #2D2D2D (Text)

## Sections

1. **Hero Section**: Animated logo, call-to-action buttons, and scroll indicator
2. **Services**: Six service cards with hover effects
3. **Projects**: Featured project showcase with overlay effects
4. **About**: Company information with animated statistics
5. **Contact**: Contact form and information

## Getting Started

Simply open `index.html` in your web browser. No build process or dependencies required!
